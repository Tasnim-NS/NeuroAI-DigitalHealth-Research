About Our Group

The NeuroAI and Digital Health Group combines the latest advancements in artificial intelligence with neurotechnology and healthcare to develop innovative solutions aimed at improving human health and well-being.

Our multidisciplinary team works on projects ranging from brain-computer interfaces (BCIs) to AI-powered tools for early disease detection and personalized healthcare solutions.

Our Current Projects

  • BCI Controlled Neurofeedback to Improve Attention and Memory Consolidation

    This project focuses on using Brain-Computer Interface (BCI) controlled neurofeedback to enhance cognitive functions like attention and memory consolidation. The goal is to improve mental performance through real-time feedback based on brain activity.

  • Machine and Deep Learning on PPG-Based Automated Sleep Stage Classification

    In this project, we apply machine learning techniques to photoplethysmogram (PPG) data for automated sleep stage classification, aiming to improve sleep tracking and related health insights.

  • AI-Based Early Detection of Parkinson’s Disease from Facial Emotion and Motor Symptoms

    We utilize AI algorithms to analyze facial expressions and motor symptoms for early detection of Parkinson’s Disease. The project aims to create a tool for non-invasive diagnosis.

  • EMG-Based Adaptive Mobile Interaction System for Visually Impaired Users: Gesture Recognition and AI-Driven Control

    This project develops an EMG-based mobile interaction system for visually impaired users, enabling them to interact with devices using gesture recognition and AI-driven control systems.
